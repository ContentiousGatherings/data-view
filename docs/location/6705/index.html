Tjugutvfi ledamöter i interimsstyrelsen, jemte da förutvarande, utsågos, och fick denna interimsstyrelse i uppdrag att utarbeta förslag till Btadgar för folkmötesföreningen, hvilka stadgar skulle framläggas på ett folkmöte, som utsattes att försiggå vid Katrineholm den 25 juni.
