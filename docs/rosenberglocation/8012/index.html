Description (Beskrivning)
Bäckaby. Sn i Vestra hd, Jönköpings län, i det höglända bergslandet österut från Stockaryds station på S. stambanan. 0,537 qv.-mil, hvaraf land 0,515 eller 11,926 tnld, 19 1/2 mtl, 1,127 inv., 386,600 kr. fastighetsvärde. Bergstrakt med skog, småsjöar och mossar; flere bergstoppar öfverstiga tusen fots höjd. Inga större gårdar eller industriella inrättningar än några små sågar och qvarnar förekomma här. 3 skolhus. Sn annex till Fröderyd.
