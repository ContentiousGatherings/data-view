Description (Beskrivning)
Bergqvara i Kronobergs län. Herresäte i Bergunda sn, Kinnevalds hd, vid Bergqvarasjön och Heligeån, nära Räppe jernvägsstation. Det gamla godset består af Bergqvara säteri 5 mtl, Ask 2 1/4, Askummetorp 1 1/2, Bergunda 4, Björketorp 1/4, Kyletorp 1/4, Långstorp 2 5/8, Lönås 1/4, Marklanda 3, Påtetorp 1, Qvälleberg 1/4, Ramdala 1/2, Rockatorp 3/4, Räppe 1/2, Sevedsäng 1/2, Vestenhaga 1, tills. 23 5/8 mtl (eller största delen af sn), tax. 430,500 kr., samt följande betydliga verk: vid Räppe: qvarn, såg, barkstamp, vadmalsstamp, ölbryggeri, frökläckningshus, till. tax. 34,000 kr.; vid Bergqvara: Örsleds qvarn, Bergqvara linoljeslageri, manufakturverk och stärkelsefabrik, tills. tax. 27,000 kr. Vidare lyda under Bergqvara 2 mtl Öjaby i grannsocknen Öjaby, tax. 41,500 kr.; det hela således uppskattadt till 533,000 kr. Egaren af Bergqvara är patronus till Bergunda pastorat.
