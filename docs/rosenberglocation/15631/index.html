Description (Beskrivning)
Gistad i Östergötlands län. Sn i Åkerbo, Bankekinds och Skärkinds hd, vid Ö. stambanan och jernvägen Norsholm-Vestervik, af hvilka den förra har vid landsvägen Norrköping-Linköping stationen Gistad, 19,8 mil från Stockholm, 37,9 mil fr. Malmö, tillika post- och telegrafstation. 0,207 qv.-mil eller 4,783 tnld, 32 5/8 mtl, 856 inv., 780,740 kr. fastighetsvärde. (Bankekindsdelen: 2 mtl tillhöra Örtomta jordebokssn; Åkerbodelen: 1/2 mtl föres till Törnevalla sn.) Kuperad mark med skog; åkerbruk är ortens näring. Större gårdar äro Hennebjörke, Harstad och Äleby. Vid Kumla nära jernvägsstationen finnes gästgifvaregård och Skärstads hds tingshus. Sn utgör ett konsist. pastorat af 3 kl. i Linköpings stift.
