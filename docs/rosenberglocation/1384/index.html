Description (Beskrivning)
Aspeboda. Sn i Kopparbergs och Aspeboda tg, s.v. om Falun och n.o. om Dalelfven; Bergslagernas jernvägsstation Ornäs ligger strax utom s. gränsen. Landsvägen Falun-Leksand går genom sn. 0,781 qv.-mil, hvaraf land 0,740 eller 17,137 tnld, 1,199 inv., 835,879 kr. fastighetsvärde. Skogstrakt med jordbruk till hufvudnäring; förnämsta bygden finnes utefter en å, som för de inom sn belägna sjöarne Stora och Lilla Aspans vatten sydost ut till sjön Runn. Af industriella inrättningar märkas Valsfors masugn och manufakturverk; vid Vassbo finnes länets landtbruksskola. Vid Smedsbo är gästgifvaregård. Sn är annex till Torsång, men skall vid vederb. komministrars afgång särskiljas till ett eget konsist. pastorat. Aspeboda, bekant från Gustaf Vasas dagar, hade förr många koppargrufvor och hyttor, nu alla nedlaga.
