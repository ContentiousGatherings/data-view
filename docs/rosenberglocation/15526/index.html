Description (Beskrivning)
Gimmersta i Södermanlands län. Herregård i Julita sn, Oppunda hd, vid sjön Öljaren strax s. om kyrkan. Egendomen omfattar i Julita sn: Gimmersta 4 mtl, Viksäter 1, St. Danviken 1, Hillersta 19/36, Bränntorp 1/4, Buttorp 1/4, Hedkarstorp 1/2, Hvithälla 1/2, Hagtorp 1/8, Kärr 1/2, Råby 1, Strand 1/2, Dalbol 1, Espedal 1/8, Fallet 1/4, Grabbetorp 1/2, Gölet eller Sörtorp 1/8, Gröndal 1/8, Hökhult 1/8, Källtorp 1/8, Krämbol 1/8, Leby 2, Råmbol 1, Råd 1/2, Rensbol 1/8, Stenstorp 1/8, Stenkulla 1/4, Stubbetorp 1/8, Skogstorp 1/8, Svalnäs 1/2, Torp 1/2, Träda 1/4 och Åsen 1/2, tills. 18 47/72 mtl, tax. 410,600 kr., Gimmersta 2 qvarnar 6,000 kr., såg 3,500 kr,m kalkugn 200 kr., tegelbruk 6,800 kr., frälseränta 14,944 kr.; i Floda sn; Norrdal 1/4 mtl, Nygården 1/8, Rudäng 1/4, Simonstorp 1/4, Skirtorp 1/8, Säbäcken 1/4, Västorp 1/8, Vattugluggen 1/2, Åkersäter 1/4, Ängtorp 1/2, Ingvallsboda 1/2, Bresäter 1/4, Fagernäs 1/4, Gummetorp 1, Hissjö 2, Hvittorp 1, Hökkärr 1 och Hällvik 1, tills. 9 5/8 mtl, tax. 242,800 kr.; i V.
