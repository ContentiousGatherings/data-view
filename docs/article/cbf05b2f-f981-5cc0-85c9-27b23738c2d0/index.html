öfrige framstående och för reformen nitälskande män för att utfärda inbjudningen till folkmötet som hölls sistlidne söndag i den vackra parken vid Attarpi ten jernvägsstation vid Det var naturligt att det vigtiga ämnet så allmänt öfverklagadt skulle locka en massa menniskor att deltaga i och afhöra de tal som vid detta tillfälle komme att hällas Mötet var utsatt att börja klUckan _ på eftermiddagen
