Första majdemonstrationen i år var imponerande genom sin, efter förhållandena, stora tillslutning från arbetarnfes sida. (Visserligen Var demonstranternas antal såget mindre i år än vid majdemonstrationen i fjor, men örn man betänker att arbetsstyrkan minskats, så väl vid verkstäderna .sorn vid stadens egna arbeten, med sammanlagt ett 200-tal, så kan man tryggt påstå att tillslutningen Var relativt större i år än vid något föregående. Äfven kan sägas, att deltagarne utan undantag utmärkte sig för ett i allo värdigt och nyktert uppträdande. Något ovanligt inträffade i år den 1 maj, ovanligt åtminstone för vårt samhälle, emedan det var skådeplats för tvenne demonstrationer den dagen Goodtemplarna hade nämligen tagit sig för ia$t med fenor oell musik tåga ge nom stadens liufvudgator, hvilket sked de så tidigt som kl. half 6 f. m. Delta game, som utgjordes mestadels af kvin nor, uppgick till ett 300-tal. Kristian stadsbladet sorn omnämner demonstra Ilonen påpekar på tal om antalet del tagare, att dessa i stället borde upp gått till 1,000. (Det lär nämligen finnas så många goodtemplare i staden). Det var alltså 700 som strejkade. Månne dessa inte ansågo demonstrationen lite felaktigt anordnad. Först och främst håde goodtemplarna denna gång, såväl som mången gång förut, helt och hållet ignorerat öfriga nykterhetssällskap, hvilka icke ha så få medlemmar, och om det skulle demonstreras mot Bachus så hade väl dessa bort inbjudas. För öfrigt torde det ha varit bättre örn de med sina fanor deltagit i demonstrationen för medborgarrätt och 8 timmars arbetsdag, hvilka reformer äfven ur nykterhetssynpunlct torde vära vårda att demonstrera för.
