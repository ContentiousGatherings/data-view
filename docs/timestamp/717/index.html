Excerpt
Stockholms centralstation, vid norra stationen och lokomotivstallarne äfvensom af vid centralstationen stationerade stadsbud bildades den 11 d:s. Föreningens ledamöter förbinda sig att ej besöka utskänkningsaktiebolagets utskänkningsställen eller försäljningsmagasin. Mötet var besökt af circa 150 personer, ooh inbjudning till det samma hade utfärdats af 3 vid centralstationen anstälda stationsbetjente. Deltagarne visade sig i allmänhet myoket intresserade för mötets ändamål. Etter diskussion, som varade ett par timmar, skreds till val at en komité, bestående af 5 personer, för utarbetande af förslag till stadgar. Åt komitén uppdrogs att så fort ske kunde utarbeta det samma samt att utlysa ett nytt möte för att behandla förslaget för att välja ordinarie styrelse. Mötet afslutades kl. half 1 på natten, och man skildes åt med de bästa förhoppningar
