Offentliga möten i Boulognerskogen

✓ Valid

Meeting Type Details

Raw Name
Offentliga möten i Boulognerskogen
Description
Offentliga möten i Boulognerskogen för att visa sympatier för de strejkande och för att demonstrera för föreningsrätten och för tvistens biläggande genom skiljedom.
Excerpt
maktmissbruk och godtycke, sorn polismyndigheterna här gjort sig skyldiga till sedan strejken utbröt, är polismästarens förbud mot hållande af offentliga möten i Boulognerskogen. Detta är den enda större offentliga plats inom stadens område som lämpar sig för offentliga möten och som allmänheten har fritt tillträde till densamma, har den förut användts såväl vid de årliga 1-maj-demonstrationerca som vid tillfällen, då strejker varit rådande, då stora grupper af arbe tare här dagligen haft Bina möten t. ex. när brädgårdsarbetarestrejk egde rum och likaså när etufveriarbetarne voro i konflikt.
Extraction Reasoning
Det är offentliga möten.
