Meeting Type Details

Raw Name
Folkmötena mot revisionismen
Description
Folkmöten som hölls på senhösten 1895 i olika trakter av landet för att ta avstånd från revisionismen.
Excerpt
De folkmöten som på senhösten 1895 i olika trakter af landet togo afstånd från den högtrafvande Kuddbyresolutionen och liknande revisionisttokerier hade dock större betydelse än något enskildt författarskap såsom mätare på svenska folkets känslor i dåvarande "norska fråga"
Extraction Reasoning
Folkmötena beskrivs som en reaktion från svenska folket
