Meeting Type Details

Raw Name
Möte om utträde ur arbetarkommunen
Description
Möte för att diskutera ett förslag om utträde ur arbetarkommunen i Arlöv.
Excerpt
Vad andans man de äro, kan man förstå, då de i agitationen för sitt förslag begagna sådana argument som att avdelningen skulle spara in avgiften till kommunen. Vi hoppas, att Arlövs träarbetare, då frågan på torsdag kväll kommer till behandling, tydligt säga ifrån sin mening till dessa element, som intet högre önska än spränga arbetarorganisationerna.
Extraction Reasoning
Det är ett möte för att diskutera ett förslag om utträde ur arbetarkommunen.
