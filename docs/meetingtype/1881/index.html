Meeting Type Details

Raw Name
majdemonstration
Description
En demonstration där kvinnor uppmanas att delta och gå med i tåget.
Excerpt
Dubbel skyldighet att verka för livlig tillslutning till majdemonstrationien från kvinnornas sida ha de kvinnliga partivännerna i år. Det vore skam, örn kvinnorna då ej skulle samlas till demonstration, gå med i t åget i stället för att stå på trottoarerna eller vid vågarna och titta på helt oförstående, som ofta är kvinnors sed.
Extraction Reasoning
Texten nämner 'demonstration' och 'tåg' vilket tyder på en protest eller folkmöte.
