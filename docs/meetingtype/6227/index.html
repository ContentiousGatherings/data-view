Stort folkmöte utan politik. Till Dagens Nyheter skrifves; Det förut i tidningarna anmälda folkmötet i Oxelösund hölls i söndags och gynnades af ett särdeles angenämt väder. Tidigt på morgonen afgick särskildt för mötet från Eskilstuna anordnadt extratåg, som ju närmare det kom Oxelösund blef allt mer och mer öfverfyldt af lustfarande. Dessutom anlände från Norrköping ångarne Norden och Östern som medförde ett försvarligt antal Norrköpingsboar, bland dessa medlemmarne af nämde stads arbetareförening, så att omkring 4,000 (ja många vilja uppskatta det till 5,000), personer voro församlade »blot till lyst», ty det var ej fråga om politik utan blott att på bästa vis kunna roa sig i det fria.
