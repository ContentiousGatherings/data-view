Folkmötet på Ladugårdslandet

✓ Valid

Meeting Type Details

Raw Name
Folkmötet på Ladugårdslandet
Description
Komiterade för folkmötet på Ladugårdslandet har organiserat ett möte för att diskutera bostadsbristen och har skickat en skrifvelse till stadsfullmäktige.
Excerpt
Komiterade för folkmötet på Ladugårdslandet hafva genom öfverståthållareembetet ingifvit en skrifvelse till stadsfullmäktige, hvari de anhålla, att staden dels måtte, åt gådane personer, som den 1 oktober saknade bostad, upplåta möjligen lediga i stadens hus befintliga rum med eldstäder, dels för sådant ändamål låta insätta tillfälliga •ldstäder i dertill lämpliga, men i saknad af sådana varande rum.
Extraction Reasoning
Det är ett folkmöte organiserat av komiterade för att diskutera bostadsbristen.
