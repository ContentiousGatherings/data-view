Handtverkaremöte

✓ Valid

Meeting Type Details

Raw Name
Handtverkaremöte
Description
Möte för handtverkare i Drammen där beslut fattades om att tillsätta en kommitté för att utarbeta förslag till lagstiftning.
Excerpt
På ett i fjol i Drammen hållet handtverkaremötp der dessa äsigter voro nästan enhälliga beslöts det att tillsätta en komité som skulle utarbeta förslag i denna riktning förnämligast till lagstad gandnn hvilka skulle inlemmas till stortin get.
Extraction Reasoning
Detta är ett möte för handtverkare där beslut fattades och en kommitté tillsattes.
