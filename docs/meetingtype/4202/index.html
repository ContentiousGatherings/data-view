Meeting Type Details

Raw Name
Stormigt folkmöte i Jyderup
Description
Högermöte och venstermöte i Jyderup, Danmark, som ledde till ett blodigt slagsmål.
Excerpt
Stormigt folkmöte. I söndags hölls i Jyderup i Danmark ett högermöte, som var besökt af omkring 5,000 deltagare. Samtidigt hölls på samma ställe ett venstermöte, vid vilket ungefär hälften så många deltagare infunnit sig. Vid båda mötena fäldes skarpa yttranden. Slutet på allt sammans blef ett blodigt slagsmål.
Extraction Reasoning
Händelsen beskrivs som ett 'stormigt folkmöte' med högermöte och venstermöte som ledde till våldsamma sammanstötningar.
