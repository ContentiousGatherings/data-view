Raw Name
Franskvänlig demonstration
Description
En väldig menuiskomassa fylde gatorna från prefekturen till legationsbyggnaden. Beridna gendarmer skingrade folkhopen. Tre gendarmer blefvo sårade. Utanför franska legationen var samlad en talrik polisstyrka for att hålla till baka demonstranterna. Der häktades 23 personer, bland dem redaktören för den republikaneka tidningen El Ideal, hr Emilio Prieto.
