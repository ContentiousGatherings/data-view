Raw Name

Description
Arbetare i Landskrona planerar att demonstrera.
Excerpt
För öfrigt ha vi här en stadsfiskal och en öfverkonstapel L., som nog kunde förtjäna sitt eget kapitel. Örn arbetarne här någon gång ha möte eller annan tillställning, så ha dessa herrar brådt med att operera, referera, spionera och trakassera, detta senare gent emot dem som våga upplåta plats åt socialisterna. Nog af: någon demonstration fingo vi altså inte. Hvarför en del organisationsmedlemmar drogo sig för att gå med var därför att de tviflade på stor tillslutning nog för att demonstrationen skulle bli imponerande.
