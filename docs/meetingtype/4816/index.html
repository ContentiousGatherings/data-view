Raw Name
Folkmöten
Description
Folkmöten där vännerna av indelningsverket och nihilisterna åstadkommer ett väsen.
Excerpt
Han förutsåg att vännerna av indelningsverket i förening med nihilisterna då skulle på folkmötena i landet åstadkomma ett sådant väsen att andra kammaren ansåge sig böra göra ytterligare nedprutningar och derigenom ge första kammaren en ännu bättre anledning att avslå ett sådant förslag än den hade 1883.
