Description

Excerpt
Folkmöte, Enligt telegram från Ystad har i lördags ett folkmöte hållits ined de valberättigade invånarne i Färs härad. Omkring 700 personer hade infunnit sig. Hemmansegaren Martin Trulsson i Starrarp, som hittills varit valkretsens representant i andra kammaren, uppstäldes å nyo som kandidat Hr Trulsson förklarade sig icke obenägen att mottaga återval. Derefter diskuterades några politiska frågor, hvarvid mötet beslöt &tt som sin åsigt uttala : att skattebördorna skola fördelas lika, d. v. s. efter hvars Och ens skatteförmåga; att försvaret borde göras billigt och att rikets neutralitet samt mellanfolkliga förlikningsdomstolar vore önskvärda.
