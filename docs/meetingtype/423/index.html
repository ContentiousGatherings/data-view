Meeting Type Details

Raw Name
Möten för att diskutera Göteborg-Jönköpings-banan
Description
Möten för att diskutera Göteborg-Jönköpings-banan och dess lämplighet och nödvändighet.
Excerpt
Möten för utbyte af åsigter rörande denna föreslagua jernväg, eller kanska hufvudsakligen för meddelande uf upplysningar rörande densammes lämplighet eller nödvändighet, hafva hållits i Jönköping och i Månstad (Kinds härad). Till i går hade landshöfdingen grefve E. Sparre shinmankal'at dylikt möte till Borås (för Borås stad, Ås, Marks, Vedens och Bollebygds härader), och till i dag har grefven ut yst sammanträde i samma syfte å Göteborgs börs nu d de intresserade i Göteborg, Askims och Säfvedals härad.

Texten nämner möten för att diskutera Göteborg-Jönköpings-banan.
