Meeting Type Details

Raw Name
Folkmöten och folkomröstning.
Description
Folkmöten och folkomröstning om allmän rösträtt och progressiv beskattning.
Excerpt
Här ha hundratals folkmöten förklarat att de ej ville vara med om nya bördor, med mindre allmän rösträtt och progressiv beskattning först lemnades. De na icke som jordhrukarne begärt rättelse i gamla orättvisor, blott att de nya bördorna skulle bli billigt fördelade. Des tom ha öfver 200,000 mvnlige män uttalat sig för folkriksdag. Det har varit ert verklig folkomröstning, som tydligt angifvit folkets önskningar.
