Kakelngnsmakareförb. affi. i malmö hade extra sammanträde den 25 dennes, hvarvid protokollet från föregående mötet upplästes och godkändes. Herr C. Löfgren valdes att sälja festtecken vid l-majdemonstrationen. Bref från Kalmar upplästes, af vilket framgick, att strejk utbrutit därstädes, föreningen ansåg sig dock böra afvakta närmare upplysningar, innan något göres för stridens afhjälpande. Vidare redogjorde komitén för sitt arbete rörande underskrifvandet af vår prislista. Det upplystes att mästarne voro villiga underteckna densamma. Därjämte beslöt föreningen att icke till hjälp använda sådana personer, som ej tillhöra Sedan några smärre frågor blifvit behandlade, afslutades mötet.
