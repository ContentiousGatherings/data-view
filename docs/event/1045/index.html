Vi lämna emellertid nu prästerna åt sitt öde och öfvergå till saker, som mera beröra arbetarne och deras intressen. Som Arbetets läsare ha sig bekant Ängö vi ingen majdemonstration till stånd här i år. Det kändes bra bittert för en klassmedveten arbetare, då; nästan hela världens organiserade arbetare på sina respektiva platser samlade sig till demonstrationer för 8timmarsdagen — här i Sverge äfven för den allmänna rösträtten — att inte få rycka fram med i ledet för ett par mänskliga rättigheter, som arbetarne för länge sedan borde haft.
