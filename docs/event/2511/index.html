Vid majdemonstrationen i Sölvesborg skall insamling företagas till förmån för agitationen bland landsbygdens proletärer. På enstaka platser och vid enstaka tillfällen kan man ju vidtaga denna åtgärd. Helst som vid vira demonstrationer så mycket folk är på benen, som sillan bidrager med ett öre till omkostnaderna för propagandan men kanske ändå har litet direkt nytta av rörelsen. Hur göra för övrigt alla andra rörelser, sekter och partier för att få medel till sin verksamhet 1 I år ha vi ju även offren för arbetsköparnas lockoutraseri att hålla om ryggen samt de många eljes arbetslösa. Men härmed ha vi inte alls sagt, att det skulle tagas upp »kollekt» vid majdemonstrationen i Malmö, ty det har gjorts 'rätt ofta förut. Vi mena baxa, att saken med viss urskillning går an. »I stora ting enighet, i små ting frihet, i allt förstående kärlek», heter det.
