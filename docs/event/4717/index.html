I vårt senaste torsdagsnummer anstälde vi några betraktelser med anledning af det sätt hvarpå den svenska tidningspressen uppfattat folkmötena och specielt det möte som sista lördagen i juni månad hölls i Katrineholm. Vi framhöllo särskildt den ovärdiga skildring af detta möte som Göteborgs-Posten meddelat samt påpekade dervid kontrasten mellan nämde tidnings referat af mötets öfverläggningar och det omdöme som dr Charles Dickson vid mötet fällde öfver det sätt, hvarpå dessa öfverläggningar blifvit förda — en kontrast som vi funno desto mera öfvertaskande som vi hört uppgifvas att dr Dickson vore delegare i nämnde tidning.
