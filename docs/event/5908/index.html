Arets majdemonstration var det ståtligaste, som dittills försiggått på platsen, med stor anslutning från de rundt örn liggande landtgårdarnes organiserade arbetare. Redan tidigt på morgonen var Svedala musikkår på benen, och inblåste den stora mönstringsdagen, hvarefter den afreste till Holmeja, Yddinge och Börringe för att afhämta dessa platsers demonstrationsdeltagare. Vid middagstiden hade alltså mycket arbetare sammandragits, så att tåget, då det vid 3-tiden satte sig i marsch genom Svedala storgata, räknade öfver 1,000 personer och tog sig helt imponerande ut med sina sju fanor och ett standar i téten. Tribunen var placerad på torget, och talet hölls af fru Elma Danielsson, som, mot bakgrunden af den bittra sanningen »Och' ropet var alltjämt detsamma: Bröd, Och svaret var alltjämt detsamma: Stenar» —
