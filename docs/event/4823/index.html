Riksdagsmannen för Yästra härad i Småland, hr J. Sjöberg, har i skrifvelse till ordförandena i kommunalstämmorna inom valkretsen uttalat dien önskan att de olika kommunerna måtte på lämipligt sätt sättas i tillfälle att mera samifälldt uttala sig angående det nya härordningsförslaget samt att han sedan måtte få del af de åsikter som i nämnda fråga göra sig gällande. Hr Sjöberg uttalar den. meningen att man på detta sätt bättre få reda på hvad allmänheten tänker genom ett gemensamt större folkmöte. Riksdagsman Sjöberg uttalar •in egen åsikt på följande sätt: För min del anser jag förslaget om ettårig värnplikt såväl för den enskilde som för landet i dess Helhet alltför betungande.
