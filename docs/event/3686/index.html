Vid valmöte i Halsberg i måndags uttalade man sig för att den blifvande riksdagsmannen för Kumla—Sundbo härad i Nerike bör vara jordbrukare nykterhetsvän sparsam vid anslag synnerligast de militära vän af utjämning af jordskatterna samt vän af den kommunala rösträttens begränsning till högst V20 af kommunens röstetal Rörande den blifvande riksdagsmannens hållning i tullfrågan voro önskningarna delade i det ena hälften af valmännen önskade sig en man som arbetar för införande af skydd åt jordbrukets alster medan den andra hälften betackar sig för en sådan representant Ett par talare af frihandelsvänlig bekännelse uttalade tillika önskvärdheten af att den blifvande riksdagsmannen ville verka för nedsättning af det politiska strecket
