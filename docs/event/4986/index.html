Sistlidne lördags afton af hölls ett gemensamt sammanträde, hvarvid kongressombudet lämnade en redogöralse för de viktigaste punkterna af kongressens förhandlingar. Därefter valdes interimskretsstyrelse att fungera till den 1 juli samt att uppgöra ett förslag till stadgar för den lokala organisationen. Slutligen behandlades l:ste majdemonstrationen. Mötet beslöt efter moget öfvervägande att ej söka anordna någon demonstration den 1 maj af följande skäl: För det första riskerade man att ej vinna tillslutning i sådan grad, att demonstrationen komme att efterlämna det intryck uppåt sorn vore önskligt.
Extraction Reasoning
Ett sammanträde nämns där kongressombudet lämnade en redogörelse och en interimskretsstyrelse valdes.
