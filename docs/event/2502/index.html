Ar 1848 blef äfven för rumänerna ett år af frihetsrörelse. Ett af 40,000 män bevistadt folkmöte på frihetsafältet vid Blaj (Blasendorf) antog en resolution, hvari uttrycktes att rumänerna borde genom grundlag erkännas såsom sjelfständig nation, att den grekisk-ortodoxa religionen, gom flertalet af dem bekände, borde likställas med öfriga kyrkor, att näringsfrihet, yttrandefrihet, församlingsfrihet och jurydomstolar borde införas samt att ett förslag om Transylvaniens införlifvande med Ungern borde uppskjutas tills rumänerna blefve vederbörligen representerade i riksförsamlingen.
