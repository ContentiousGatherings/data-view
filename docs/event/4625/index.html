Malmö arbetsmanuafackförening hade sammanträde söndagen den 3 d:s: En skrifvelse var inlämnad från Malmö arbetsmannafaokförenings sjuk- och begrafningskassa om att låta personer, stå ende utom fackföreningen, få ingå i sjukkassan, men dock tillhöra någon organisation. Härvid yttrade sig flera talare både för och emot förslaget. Sjukkassans stadgar säga, att ingen får tillhöra sjukkassan utan att först vara medlem i arbetsmannackföreningen. Frågan bordlädes till nästa sammanträde. Det beslöts på ett föregående sammanträde, att arbetsmannafackföreningen skulle tillhöra socialdemokratiska arbetarepartiet. En fullständig tablå var mi upprättad öfver medlemsantalet, af hvilken framgick att endast 47 fasta medlemmar tillhöra Malmö arbetsmannafackförening. 5 nya medlemmar inskrefvos i föreningen. Föreningen beslöt sig för att uttala sin skarpaste protest gentemot Berghells utvisning och antog följande resolution: Malmö arbetsmannafackförening uttalar sin skarpaste protesc gentemot Berghells utvisning, så mycket mer som Berghell ej gjort sig skyldig till något brott, utan endast uttalat sin förkastelsedom öfver dogmer, som de herskande ej ens själfva tro på. Dessutom tillätes det andra utlänningar, som äro i besittning af kapital, att på det mest hänsynslösa sätt plundra sina arbetare, utan att det minsta göres ifrån regeringens sida för att förhindra detsamma, hvarför föreningen betraktar denna utvisningsdom såsom ett det skamligaste uttryck för klassvåldet, som någonsin kan existera.
