I Tynderö (Medelpad) hölls i söndags det första folkmötet på denna plats, vilket förklarade att det nya arméförslaget kräfde större utgifter än landet kan bära samt att jordbrukare och arbetare måste bestämdt uttala sig emot det samma. Dessutom antogs en rösträttsvänlig resolution, i hvilken detheten »Endast genom införandet af allmän rösträtt kan det påståendet, att arbetaren ej har något att forsvara, häfvas och äfven den nu rösträttslöse komma att känna sig som samhällsmedborgare»
