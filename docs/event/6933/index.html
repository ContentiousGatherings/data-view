Vid mötet i Katrineholms tingshus var äfven frih. Palmstierna ordförande. En ryttmästare Grönvall fungerade som sekreterare. Mötet uttalade sig för indelta arméns bibehållande, men så att kostnaderna för försvaret skulle bestridas af staten. Vidare ansågs att sjöförsvaret borde stärkas, grundskatterna utjemnas och skyddstull för jordbruksalster samt höjda tullar å lyxartiklar införas.
