Vid Handt- verkaresällskapets sammanträde onsdagen den 28 d:s diskuterades frågan om folkriksdag, som inleddes af fabrikör Olof Olsson. Intresset var enhälligt för att riksdagen måtte komma till stånd, och många talare uppträdde. Följande resolution antogs: Sällskapet anser att en folkriksdag bör komma till stånd samt erhållande af allmän rösträtt såsom ett oundgängligt vilkor för utsträckt värnpligt.
