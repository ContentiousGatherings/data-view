Arlövs arbetare uppmanas att så mangrant som möjligt infinna sig på kyrkostämman i morgon lördag kl. 4 e. m. Anslaget till Burlövs gamla kyrka skall då behandlas. Trots det ätt kyrkostämman för tre år sedan anslog 6,000 kr. och samtidigt sade ifrån, att det var det högsta belopp, som finge användas till kyrkans konservering, har kyrkorådet nu begärt ytterligare 2,300 kr.
