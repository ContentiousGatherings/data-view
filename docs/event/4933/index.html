De som inbillat sig att åtminstone landsbygden skulle vara en fristad för den af riksdagen påbjudna församlingsfriheten ha redan vid mer än ett tillfälle blifvit varnade för att hängifva sig åt dylika illusioner. Vi erinra om den åt kyrkoråden inrymda befogenheten att förbjuda »andliga» föredrag som »anses leda till söndring i kyrkligt hänseende» — en makt som någon gång användts så att förbudet följt utan föregående inkallelse inför den kyrkliga polismyndigheten. Men ännu mera betecknande är det cirkulär som för två år sedan utfärdades af en länsstyrelse till länets samtliga leronofogdar, med befallning till dem att infinna sig vid alla »folkmöten och andra dylika sammanträden der antagas kan att politiska ämnen komma att afhandlas«.
