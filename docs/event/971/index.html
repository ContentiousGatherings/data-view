Excerpt
Folkmöte hölls i Motala i lördags för öfverläggning om försvars- och grundskattefrågorna. Mötet, som var besökt af närmare 200 personer, beslöt tillställa länets riksdagsmän en adress, hvari yrkas grundskatternas afskaffande, indelningsverkets reorganisering så att staten öfvertager roteringen och mot billig afgäld torpen, samt värnpligtens utsträckande till högst 60 dagar; hvarjemte mötet med värme omfattade de af hrr Adelsköld ooh Arnoldson vid riksdagon nu väckta motioner.
