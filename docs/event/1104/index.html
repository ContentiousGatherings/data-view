Ej ens franco-russe-entusiasmen var helig for hr Tailhade, han citerade den tvärt om såsom exempel på de dumheter till hvilka den kompakta majoriteten låter sig ledas. Detta framkallade enstaka yttringar af missnöje, så ock när han talade om »den välgö rande anarkismen, som skulle skaffa oss ett bättre samhällsskick», men bifallet var hela tiden alldeles öfvervägande.
