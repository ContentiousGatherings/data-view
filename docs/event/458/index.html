I Göteborg blir likaledes tvänne demonstrationer på söndag för allmän rösträtt, socialdemokraternas och de liberales. Vid de förres lär Hj. Branting komma uppträda som talare. Rösträttsrörelsen. I söndags kväll bildades i Karlshamn af ett 20-tal personer »Karlshamns rösträttsförening».
