Folkets Ibus. För dem, som ännu inte kunnat inbetalad e förfallne terminerna på tecknade aktiebeloppet Byggnadsaktiebolaget Folkets lius, gifves tillfälle att i morgon fullgöra denna skyldighet, i det nemligen byggnadsinterimstyrelsen å »Arbetets» redaktion kl. 9—12 midd. anordnat en efteruppbörd i likhet med som förut gjorts. Vid samma tillfälle kan äfven teckning af nya aktier ske. Det är att hoppas att så väl enskilda som föreningar, hvilka ännu icke inbetalt de förfallne terminerna, i morgon göra detta. Vi ha oss bekant att styrelsen i dagarne kommer att påbörja byggnadsarbetet och då Önnes all anledning att enhvar inbe talar hvad han genom teckning af aktier förbundit sig till. Se vidare annonsen å första sidan dagens nummer.
