Vid Iivinnliga diskussionsklubbens senaste ordinarie sammanträda beslutades att deltaga i sta majdemonstrationen samt utsågs tidnings- oell märkesförsäljare. En skrifvelse från soc.-dem. ungdomsklubben örn att handla i dess affärer föredrogs, och beslutades med anledning häraf att uttala sin sympati för närnda affärer, samt att sä vidt möjligt gynna dem genom att där göra uppköp. Beslöts anordna ett offentligt möte å Malmö nya arbetareförening för behandling alf frågan örn. den förtviflade handling, som mången gång hjälplösa barnaföderskor begå oell hvars följder nyligen drabbat en olycklig kvinna. Till att representera klubbens aktier i tryckeriaktiebolaget Framtiden återvaldes de afgående.
