Excerpt
JEtt folkmöte hölls i lördags, berättar Nya Vermlandstidn ingen, i Glumseruds missionshus på inbjudan af riksdagsmannen O. Andersson i Hasselbohl. Mötet som var besökt af omkring 150 personer, öppnades med ett föredrag af inbjudaren, deri han redogjorde för den förflutna riksdagens förhandlingar och angaf sin ställning till vigtigare frågor som varit å bane. Vidare uttalade sig mötet för streckets sänkning till 500 kr. samt ansåg att tiden var inne för edernas borttagande och utbytande mot löften.
