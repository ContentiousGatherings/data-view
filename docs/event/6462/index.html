Demonstrationståg till Lilljans. Stockholms arbetarekommuns styrelse hade begärt tillåtelse att efter om fredag afton hållet folkmöte vid Lilljans i norska frågan anordna ett demonstrationståg med musik och fanor från Lilljans till Norra bantorget. Tåget skulle passera Engelbrekts-, Birger Jarlfc, Mästeraamuels, Drottningooh Barnhusgatorna. Polismyndigheterna lämnade i går på middagen tillstånd till tåget.
