Nikolai och Jakobs folkskolebarn — de två öfversta afdclningarnas Såväl flickor som gossar — inbjudna att åse ett af ritläraren Nilsson anordnadt skuggspel i Klara skolas gymnastiksal, hvarvid förekom bland annat slaget vid Helsingborg — »Stenbocken och getterna». Aftonen tillbragtes för öfrigt med lekar, dans, sång och deklamation och den stora salen var festligt upplyst med två prydliga julgranar.
