Mötet var besökt af ett par hundra personer, som uttalade sitt lifliga intresse för saken och tillsatte en komité för att utarbeta stadgar etc., hvarefter fråga* skulle vidare behandlas på ett nytt möte. Vid mötet föredrogs ett cirkulär till Stockholms arbetare, riktadt mot dervarande utekärkningsbolog, och redaktör Westenius höll ett tal, hvari han uppgå!', att Malmöbolagets årliga administrationskostnader och vinst pä sprithandeln uppgingo till 210- å 220,000 kr, som hufvudsakligen togos nr det fattiga folkets fickor.
