Ett folkmöte hölls i lördags i Delsbo Derom meddelas på telegrafisk väg följande: På inbjudning af häradshöfding P. Staaff egde härstädes i lördags ett folkmöte rum. Till ordförande utsågs häradshöfding Staaff. Mötet, afsedt för Norra Helsingland, räknade 200 deltagare, bestående af ståndspersoner, allmoge, ortens riksdagsmän, kontraktsprosten Landgren, ledamoten i Första kammaren Bruun m. fl. Mötet uttalade sig för grundskatternas bibehållande — detta beslut fattades med betydlig röstöfvervigt —, för skjutsningsskyldighetens öfvertagande af staten, för skärpta skogsafverkningslagar, för byggandet af två norrländska statsbanor från Hybo samt för nyttan af tvärbanan Näsviken—Sveg. Rörande försvarsfrågan voro meningsolikheter rådande; men allas önskan var att erhålla ett starkt försvar. Öfverläggningarna voro lifliga.
