Årsskiftet hade lockat ut till promenad ofantliga människomassor, alldeles som vid en majdemonstration. Kyrkan var fullproppad med folk strax efter kl. 10, så att de som kommo närmare kl. li fingo vackert stanna utanför, där snart trängsel uppstod. Ända till ut på »småtimmarne» pågick strömmen å gatorna, ehuru föga anmärkningsvärdt var att skåda. Emellertid framgick det oförtydbart att, örn vi haft vårt Folkets hus med dess stora, två tusen personer rymmande festsal i ordning skulle ett stort agitatoriskt slag kunnat slås. Pär Ax.
