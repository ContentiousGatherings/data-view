Norrländska jernvägsfrågan. Ett folkmöte i Wifsta, utlyst af riksdagsmannen Tjernlund, förklarade enhälligt, att 'jernvägar äro för Norrland af största vigt och i högsta grad af bohofvet påkallade' samt godkände enhälligt riktningen till Sollefteå. Skogsfrågan, alla norrländska mötens stora tvisteämne, afgjördés på folkmötet i Wifsta, efter en märklig diskussion och genom votering med 55 röster mot 52, till förmån för dem, som icke(!) vilja medgifva staten någon som helst kontroll öfver enskildas skogar.
