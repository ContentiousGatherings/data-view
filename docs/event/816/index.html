Den irländske parlamentsledamoten Parnell står, såsom redan nämts, i spetsen för agitationen och bereser hela landet. I förra veckan höllo 500 irländske förpaktare ett möte, vid vilket de öfverenskommo att icke betala någon arrendeafgift, förr än godsegarne gått in på en nedsättning i proportion till prisernas fall. På ett stort folkmöte, som ungefär samtidigt egde rum, höll en f. d. fenisk straffånge vid namn Davitt ett upproriskt tal. Han uppmanade de närvarande förpaktarne att trotsa myndigheterna, hvika gjorde gemensam sak med godsogarne, och anmärkte i anledning af de truppförstärkningar, som blifvit förda till Irland, att folket icke behöfde frukta för "soldater, som fått stryk af zulu-kaffrerne". De engelska tidningarne äfvensom de engelskt sinnade bladen i Irland yrka att ett snart slut skall göras på agitationen samt att ledarne skola ställas inför rätta. Det officif sa bladet Daily Express, som utkommer i Dublin, säger att de, som predika revolution, stå med sina lif i ansvar för det blod, som kan komma att flyta. London-tidningen Globe skrifver: "De personer, som på detta sätt oroa landet och tända brandfacklan få skylla sig sjelfva, om de skörda storm efter att hafva sått vind. Ingen regering kan förhålla sig overksam, under det att en befolkning, som påverkas af läran om att "egendom är stöld", försöker att behandla godsegarnes egendom såsom om den vore deras egen." Uti en korrespondens från London säges det vara alldeles afgjordt att regeringen skall uppträda med den största kraft gent emot hvarje försök att genomföra de läror, som predikas på förpaktarnes möten.
