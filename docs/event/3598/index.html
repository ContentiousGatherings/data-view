Föreningen Vänfast hade i går afton ett talrikt besökt sammanträde, vid vilket diskuterades följande två frågor: Ar den socialistiska verksamheten nyttig för vårt land och våra arbetare? samt Hvad hafva vi att lära från utlandet rörande denna verksamhet? Med stöd af hvad under diskussionen om båda dessa frågor af olika talare yttrades beslöt föreningen att frågorna skola så besvaras, att den socialistiska rörelsen är ytterst skadlig för de svenska arbetarne, hvarför ock föreningen för sin del icke kommer att på något sätt drifva eller gynna socialistiska läror. Samma frågor komma att behandlas på ett möte om lördag, till vilket utsedde delegerade för banbetjeningenå söder skola inbjuda sina kamrater vid Norra, Central- samt Liljeholmens stationer.
