Vid Litslena gästgifvaregård hölls den 16 auguiti ett folkmöte med Trögds, Asunda, Håbo och Bro härad, utgörande Uplands södra domgaga. Mötet var utlyst af val kret, *ens representant under förra perioden, hr i Jan Erik Andersson i Säby, hvilken afven valdes till ordförande. Det stora antalet deltagare, nästan uteslutande allmogemän, kan uppskattas till 250.
