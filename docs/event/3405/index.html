Hernösand den 21 februari. Vid dagens folkmöte i Säbrå socken invid Hernösand, vilket var besökt af 125 personer, hufvudsakligen jordbrukare, uttalades enhälligt dan skarpaste protest mot lifsmedelstullar, såsom ytterst skadliga för flertalet af befolkningen, synnerligast i Norrland, ooh endast gagnande ett fåtal godsegare i mellersta och södra Sverge. Mötet beslöt afsända resolutionen till ortens riksdagsman, hr Nydahl, för att genom honom inlemmas till statsministern.
