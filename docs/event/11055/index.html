Inbjudna-äro Riseberga, Västra Sönnarslövs, Perstorps, Örk-lljunga, Stidsvigs, Munka-Ljungby, Åstorps och Åsbo-Fagerhults arbetarkommuner. Därefter förekom på dagordningen kooperationens mål och medel samt dess innebörd.
