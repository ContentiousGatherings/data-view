Karlskrona, tiidagen den 29 mars. Arbetareföreningen sammanträdde talrikt i fredags för att diskutera om riksdagsmannakandidat. Föreningen uttalade nödvändigheten af att medverka för valet af en person som tolkade folkets önskningar och hyste frihandelsvänliga åsigter. Stark enighet ansågs nödig, Fricks kandidatur antogs enhälligt.
