Tal. slutade med att recitera Heidenstams »Medborgarsång«, åtföljd av applåder. Ordf. Lindberg föredrog därpå partistyrelsens resolution, som antogs av 1,499 jaröster mot 1 (ett) nej, valpa följde »Internationalen". Tal. slöt med utbringandet av ett leve för världsfreden, som kraftigt besvarades av fyrfaldiga hurrarop.
